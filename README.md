# agil-rpg

Proyecto: **agil-rpg** — sistema de gestión personal con estética RPG.
Versión: v1.0 (frontend estático, LocalStorage, modo oscuro)

## Cómo usar
1. Descomprime el archivo ZIP.
2. Abre `index.html` localmente o sube todo el contenido a un repositorio en GitHub y activa GitHub Pages.
3. La aplicación guarda todo en `localStorage` bajo la llave `agilrpg_v1`.

## Estructura
- index.html — dashboard principal
- avatar.html — creación/edición de avatar
- styles/ — css (main, avatar, components)
- scripts/ — js (main control, avatar, quests, stats, inventory, storage)
- data/ — JSON con equipos y habilidades

Proyecto preparado para desplegar en GitHub Pages.
